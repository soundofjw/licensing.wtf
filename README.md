# licensing.wtf

code for website [licensing.wtf](http://licensing.wtf)

## install

runs on [harp](http://harpjs.com/).

```bash
$ sudo npm install -g harp
$ git clone git@github.com:soundofjw/licensing.wtf.git
$ cd licensing.wtf
```

## start

```bash
$ harp server .
```
